# Мой первый проект: мини-ежедневник
Это программа на Python, которая запрашивает у пользователя:
- настроение за день,
- что он выучил,
- планы на завтра.

## Как запустить
1. Убедись, что установлен Python 3.
2. Скачай файл "first_file.py".
3. Запусти командой:python first_file.py

## Используемые технологии
- Python 3
- Ввод/вывод данных
- f-строки
